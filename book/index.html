# Alien-Upload是什么

Alien-Upload是一款基于 Vue开发的文件上传组件，功能强大，可以方便的实现多图上传、预览、进度查看、图片压缩等功能。

# 具备哪些功能

  • 对于指定大小的图片进行压缩
  • 手动指定图片压缩的比例
  • 图片上传进度展示
  • 多文件上传功能
  • 图片上传数目限制功能
  • 图片展示功能
